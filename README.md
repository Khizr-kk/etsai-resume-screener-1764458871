# SkillBridge AI

Auto-generated by ETS-AI 10x.

## Idea Summary
An AI-powered resume screener for Indian fresher software engineers that analyzes resumes against job descriptions, provides a score, and highlights missing skills.

## Problem Statement
Indian fresher software engineers struggle to effectively apply for jobs, often lacking insight into how their resumes align with job descriptions and what skills they need to acquire.

## Target Persona
Indian fresher software engineer

## MVP Features
- User authentication and profile management
- Resume upload (PDF/DOCX) and parsing
- Job description input (text paste or upload)
- AI-powered resume-job description compatibility score
- Identification and listing of specific skills missing from resume but present in JD
- Highlighting of skills present in resume that match JD
- Basic dashboard to view past analyses

## Tech Stack
Python (FastAPI/Django) for backend and ML, React/Vue.js for frontend, PostgreSQL for database, AWS/GCP for cloud infrastructure, Docker for containerization.
